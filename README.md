# ClawLegion

ClawLegion is a plugin-based multi-agent collaboration system implemented with Rust, a React frontend, and plugin SDKs for Rust and Python.

## Components

- `crates/app`: the `clawlegion` CLI
- `web/`: the frontend dashboard
- `crates/sdk/dynamic-plugin`: Rust plugin SDK
- `crates/sdk/python`: Python plugin SDK

## Release

The repository publishes GitHub Releases, npm/Homebrew/Linux packages, GitHub Pages content, and SDK packages through GitHub Actions.
