# ClawLegion Python SDK

`clawlegion-sdk` helps plugin authors build protocol-first Python plugins for ClawLegion.

## Install

```bash
pip install clawlegion-sdk
```

## Included modules

- `clawlegion.plugin`: plugin metadata, manifests, and scaffolding helpers
- `clawlegion.skill`: skill interfaces
- `clawlegion.tool`: tool interfaces
- `clawlegion.agent`: agent configuration helpers

## Release policy

The package version is synchronized from the repository root workspace version.
